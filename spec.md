# DP Globals Professional Website

## Overview
A professional corporate website for DP Globals featuring company information, services, and client inquiry capabilities with a clean business-oriented design enhanced with modern interactive animations, smooth transitions, and optimized performance. The website uses React Router for navigation between separate pages with enhanced smooth scrolling and performance optimizations. The application is configured for deployment on GitHub Pages with production-ready static build optimization.
